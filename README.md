# 📦 AU Launcher - 资源托管仓库

欢迎来到 **AU-Launcher-Repo**。

### 📢 项目简介
本仓库是 **[UT-DR-AU-Launcher
]** 的官方资源存储库。主要用于存放《传说之下》(Undertale) 及《三角符文》(Deltarune) 相关同人游戏的配置文件、素材及发布版本。

> **注意：** 本仓库主要作为启动器的后端节点使用，并不建议玩家直接在此处手动寻找下载链接。为了获得最佳的游戏更新与运行体验，请通过我们的官方启动器进行操作。

---


### 🚀 开发者说明 (For Developers)

1.  **资源引用**：本项目图片资源通过 `jsDelivr` 加速，访问速度稳定。
2.  **镜像下载**：为了缓解 GitHub 带宽压力，所有大文件下载均已接入 `Cloudflare` 边缘节点加速。
